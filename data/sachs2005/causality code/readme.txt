Causal Learning Without DAGs

This is the code and datasets used to generate results for "Causal Learning Without DAGs" by David Duvenaud, Daniel Eaton, Mark Schmidt and Kevin Murphy, to appear in JMLR '09.

Download

To reproduce the figures shown in the paper, simply run produce_all_figures.m.


Included are versions of the following software packages:
BDAGL ( Bayesian DAG Learner ) 
PMTK ( Probabilistic Modeling Toolkit ) 
Minfunc ( Function minimization tool )
